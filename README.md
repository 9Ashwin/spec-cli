# comet-schema

Community-contributed workflow schemas for [Comet](https://github.com/rpamis/comet). Each schema is a self-contained bundle that defines phases, transitions, guards, and templates for a development workflow.

## Usage

Schemas are installed and managed by Comet's CLI. See the [Comet documentation](https://github.com/rpamis/comet) for details.

## Schemas

| Schema | Purpose | Status |
|--------|---------|--------|
| `full` | Complete 5-phase workflow (open → design → build → verify → archive) | WIP |
